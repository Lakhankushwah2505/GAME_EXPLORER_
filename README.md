# GAME_EXPLORER_
NEXUS Chronicles – A React + Vite powered game explorer UI that lets users browse and filter games by genre with a clean and modern interface. Frontend-only project.

<IMPORTANT INSTRUCTIONS>
  
STEP 1: FIRSTLY SETUP A SIPLE REACT+VITE APPLICATION IN A NEW FOLDER ( GUIDE : INSTALL NODE AND NPM IN YOUR SYSYTEM CHECK VERSION USING COMMANDS "node -v" and "npm -v" IN COMMAND PROMPT THEN CHANGE DIRECTORY TO FOLDER WHERE YOU WANT TO SET UP YOUR REACT APP AND RUN A COMMAND "npm create vite@latest your-app-name" or simply"npx create-vite@latest your-app-name" 
  
STEP 2: THEN OPEN YOUR FOLDER IN VScode AND IN TERMINAL RUN COMMAND "npm i" and "npm run dev" YOU WILL SEE A LOCALHOST ADDRESS FOLLOW THE LINK 

STEP 3: WOW... NO MORE STEP, YOUR REACT APP MADE IT IS A SIMPLE COUNTER APP MADE BY REACT DEFAULT

so your default counter app made let's come to development but before it we have to understand basic files in react and their purpose, i am assunming that you have at least basic knowledge of react and jsx syntax 

you'll see : 
             INDEX.HTML : THIS IS THE "ROOT" OR CAN SAY ENTRY POINT TO YOUR REACT APPLICATION
             MAIN.JSX : (INSIDE SRC FOLDER) THIS IS BRIDGE BETWEEN INDEX.HTML AND APP.JSX
             APP.JSX : (INSIDE SRC FOLDER) APP CODE FILE OR  ASSEMBLY FOR APP COMPONENTS
             INDEX.CSS : TO STYLE ENTIRE HOME PAGE
             APP.CSS : TO STYLE THE APP OR APP COMPONENTS
             REST FILE ARE ESSENTIAL NPM DEPENDENCIES AND REACT DEPENDENCIES
